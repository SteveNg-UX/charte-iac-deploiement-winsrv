# Projet Charte Template Deploiement Service & Donnée Windows Server (En cours de développement)

---

### Info :
- Une charte de Template IaC en Powershell pour deployer un/des service/s dans un/des serveur/s Windows
- Dans la même philosophie que Helm Chart et Ansible mais avec les service Windows Server en version simplifié

---

### Guide d'utilisaiton :
1. Télécharger /Cloner le projet
2. Ouvrir la console Powershell
3. lancer la commande suivant :
4. se diriger dans le dossier projet
5. une fois les modifications fait en fonction du besoin, lancer la commande suivante :
```Powershell
.\run.ps1
```

##### Alternative :
1. Télécharger /Cloner le projet
2. se diriger dans le dossier projet via l'explorateur de fichier
3. Clique-droit sur le fichier "run.ps1"
4. CLiquer sur "Exécuter avec PowerShell"

---

### Remarque :
- Les seule donnée à modifier selon les besoins seront juste les élément du dossier "data"
- Si besoin, s'aider du dossier "data_template"
